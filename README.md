# ![icon](img/logo-48.png) Flood It

A little game in the telegram bot

## How to use

Set the environment variable in .env

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run src/index.ts
```

## How to play

Click cells, flood the board with a single color.
